# Contributing to Materia Reborn

Materia Reborn is currently under active development. External pull requests are not being accepted yet.

You can still help by reporting reproducible bugs and sharing focused feature suggestions through the provided issue templates. Please avoid duplicates where possible and include screenshots, logs, Minecraft/NeoForge versions, and clear reproduction steps when relevant.

By submitting a contribution, you agree that it may be distributed under the project's MIT License.
